# Pokémon Quiz

Um quiz interativo de Pokémon desenvolvido para testar conhecimentos sobre o universo Pokémon de forma divertida e dinâmica.

## 🌐 Demo

* Site: http://pokequiz.wuaze.com/
* Repositório: https://github.com/Luhenr1que/Pokemon-Quiz

---

# 📖 Sobre o projeto

O Pokémon Quiz é uma aplicação web onde os usuários podem responder perguntas relacionadas ao universo Pokémon, colocando seus conhecimentos à prova.

O projeto foi criado com foco em:

* Interface simples e intuitiva
* Experiência divertida para fãs de Pokémon
* Aprendizado prático de desenvolvimento web
* Manipulação de lógica com JavaScript

---

# 🚀 Funcionalidades

* Quiz interativo
* Sistema de perguntas e respostas
* Verificação de acertos e erros
* Interface responsiva
* Temática inspirada no universo Pokémon
* Navegação simples e rápida

---
